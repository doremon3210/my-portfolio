# Huy Nguyen — Personal Portfolio

Personal portfolio site built with SvelteKit and Tailwind CSS.

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Development

npm install
npm run dev

## Build

npm run build
